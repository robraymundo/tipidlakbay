# 🚗 TipidLakbay

**TipidLakbay** is a smart trip cost optimizer that helps users estimate travel expenses based on **origin, destination, vehicle type, fuel efficiency, and fuel price**. It is designed to make trip planning more practical, accessible, and budget-friendly.

---

## ✨ Features

### 📍 Distance & Fuel Estimation
Calculates travel distance, fuel consumption, and estimated fuel cost based on user inputs.

### 🛣️ Smart Route Comparison
Compares multiple routes to identify the **fastest, cheapest, or most fuel-efficient** option.

### 🤖 AI-Assisted Vehicle Efficiency Lookup
Allows users to enter their vehicle model, then automatically suggests an estimated fuel efficiency value using matched vehicle data and smart text matching. This reduces manual input and improves the accuracy of trip cost estimation.

### ⛽ Smart Fuel Price Input
Allows users to input current fuel prices with suggested averages, ensuring accurate and reliable trip cost estimation without relying on external APIs.

### 🗺️ Interactive Route Map
Displays the recommended route on an interactive map, allowing users to clearly visualize the trip path between the origin and destination while improving route planning through map-based navigation support.

---

## ⚙️ How It Works

1. Enter your **starting point** and **destination**
2. Select your **vehicle type**
3. Input or adjust **fuel efficiency**
4. Input the current **fuel price**
5. Click **Calculate Trip** to estimate the total trip cost

## 🎯 Purpose

This project was built to help users make better travel decisions by providing a quick estimate of transportation costs before starting a trip.

TipidLakbay aims to make trip **simpler, smarter, and more practical**, especially during times of rising and unpredictable fuel prices.

---

## 💡 Why TipidLakbay?

Many trip calculators only give rough estimates based on distance. TipidLakbay goes further by helping users compare routes, account for fuel efficiency, and make more cost-conscious travel decisions.
